🔧 INSTALLAZIONE:

1. Estrai l'archivio:
   tar -xzf vulnerability-assessment-platform.tar.gz
   cd vulnerability-assessment-platform

2. Completa i file Python:
   I file Python completi sono forniti nella conversazione precedente.
   Copia il contenuto di ciascun file nei file corrispondenti:
   
   - config.py
   - database.py
   - scanner_engine.py
   - report_generator.py
   - app.py
   - scanners/*.py (tutti i 5 scanner)
   - templates/*.html (tutti i 3 template)

3. Esegui l'installer:
   chmod +x installer.sh
   ./installer.sh

4. Avvia la piattaforma:
   source venv/bin/activate
   python3 app.py

🌐 ACCESSO:
   http://localhost:8000
