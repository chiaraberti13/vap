# SOC 2 - Controlli di Compliance

## Security
- Autenticazione JWT opzionale e API key.
- Rate limiting per mitigare abusI.
- Header di sicurezza (HSTS, CSP, X-Frame-Options).

## Availability
- Health/readiness check per monitoraggio.
- Scheduler per operazioni di maintenance.

## Confidentiality
- Classificazione dati per scansioni e audit.
- Esportazione con anonimizzazione del target.

## Processing Integrity
- Validazione input e gestione errori controllata.
- Audit trail per le azioni critiche.

## Privacy
- Consenso esplicito per Privacy Policy e Termini.
- Export e deletion per richieste GDPR.
